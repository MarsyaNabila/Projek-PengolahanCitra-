Pada percobaan ini dilakukan deteksi pejalan kaki menggunakan metode HOG (Histogram of Oriented Gradients) dan SVM (Support Vector Machine) yang tersedia pada library OpenCV. Program membaca gambar yang berisi objek manusia, kemudian menganalisis pola gradien pada gambar untuk mengenali bentuk tubuh manusia. Setelah proses deteksi berhasil, sistem memberikan kotak pembatas (bounding box) berwarna merah pada setiap pejalan kaki yang terdeteksi. Hasil pengujian menunjukkan bahwa program mampu mengenali dan menandai objek manusia pada gambar dengan baik sehingga dapat digunakan sebagai dasar pengembangan aplikasi computer vision seperti sistem pengawasan dan kendaraan otonom.

Pada percobaan ini dilakukan deteksi pejalan kaki pada video menggunakan OpenCV dengan metode HOG (Histogram of Oriented Gradients) dan SVM (Support Vector Machine). Program membaca video frame demi frame, kemudian menganalisis setiap frame untuk mendeteksi keberadaan manusia. Objek yang teridentifikasi sebagai pejalan kaki akan ditandai dengan kotak pembatas berwarna merah. Hasil pengujian menunjukkan bahwa sistem mampu mendeteksi dan melacak pejalan kaki secara real-time selama video diputar. Deteksi dapat dilakukan pada beberapa orang sekaligus selama objek terlihat dengan cukup jelas pada video.
